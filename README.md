## 🚀 Quick Start Guide

### Step 1: Download the Website Files
Download the `skyline_premium_index.html` file from this repository.

### Step 2: Prepare the File
Rename `skyline_premium_index.html` to `index.html`

### Step 3: Upload to GitHub
1. Go to your GitHub repository
2. Click **Add file** → **Upload files**
3. Select and upload `index.html`
4. Click **Commit changes**

### Step 4: Deploy with Vercel
1. Go to [Vercel.com](https://vercel.com)
2. Click **New Project**
3. Import your GitHub repository
4. Click **Deploy**
5. Your site will be live at `https://skyline.vercel.app` (or similar)
